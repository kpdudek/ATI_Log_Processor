 JR3PCI - Version history
=========================================================

JR3PCI Drivers version 1.1.1,  2/26/15.

* Added device IDs 1111h and 1112h to the INF file.
  - Note: No actual cards were available for testing

=========================================================

JR3PCI Drivers version 1.1,  2/25/15.

* Now support 1 and 2 channel JR3PCI cards.
* Test application verifies correct functionality for card instances 1..4 and channels 0..1.
* Tested on 32/64 bit XP, Vista and 7 platforms.
